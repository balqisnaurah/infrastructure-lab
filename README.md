# infrastructure-lab

Dokumentasi eksplorasi infrastruktur cloud menggunakan VirtualBox dan Docker.

## Yang Sudah Dicoba

### 1. Setup Virtual Machine
- Install Ubuntu Server 24.04 di VirtualBox
- Konfigurasi jaringan dengan port forwarding

### 2. Docker - Nginx Web Server
- Menjalankan container Nginx
- Mengakses web server melalui browser

### 3. Docker Compose - WordPress + MySQL
- Deploy WordPress menggunakan Docker Compose
- WordPress berjalan sebagai CMS dengan database MySQL

### 4. Monitoring - Grafana
- Menjalankan Grafana di Docker
- Eksplorasi dashboard monitoring
